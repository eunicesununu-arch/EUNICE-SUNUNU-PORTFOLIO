# EUNICE-SUNUNU-PORTFOLIO
# README -- Perfume Business Website

## 1. Project Title

**Eunice Sununu Perfume Brand Website**

## 2. Introduction

This project is a web-based application developed to promote and
showcase the perfume business of **Eunice Sununu**, an Architecture
student and entrepreneur. The website provides an online platform where
customers can view available perfume products, learn about the brand,
and contact the business owner.

The project demonstrates the practical application of web development
concepts such as HTML structure, CSS styling, and basic JavaScript
interactivity.

## 3. Aim of the Project

The main aim of this project is to design and develop a simple,
interactive, and user-friendly website that promotes a perfume business
and provides customers with easy access to product and contact
information.

## 4. Objectives of the Project

The objectives of this project are: - To create a professional online
presence for the perfume business\
- To display perfume products in a clear and attractive way\
- To provide contact information for customers\
- To apply web development skills learned in class\
- To design a responsive and user-friendly interface

## 5. Scope of the Project

The scope of this project includes: - Home page for brand introduction\
- About page describing the business owner\
- Products section for perfume display\
- Contact section for customer inquiries\
- Basic front-end functionality

The project does not include online payments or a full e-commerce system
at this stage.

## 6. Tools and Technologies Used

The following tools and technologies were used: - HTML5 -- Website
structure\
- CSS3 -- Styling and layout\
- JavaScript -- Interactivity\
- Visual Studio Code -- Code editor\
- Web Browser (Chrome/Edge/Firefox) -- Testing

## 7. HTML Elements and Attributes Used

### 7.1 HTML Elements

  Element             Description
  ------------------- -----------------
                      Root element
                      Metadata
                      Page title
                      Main content
                      Top section
                      Navigation
                      Content section
                      Grouping
                      Headings
                      Paragraph
  `<img>`{=html}      Images
  `<a>`{=html}        Links
                      Lists
                      Form
  `<input>`{=html}    Input
  `<button>`{=html}   Button
                      Footer

### 7.2 HTML Attributes

  Attribute     Purpose
  ------------- ------------------
  src           Image source
  href          Link
  alt           Alternative text
  class         Styling
  id            Unique ID
  type          Input type
  placeholder   Hint
  value         Default value
  name          Field name
  style         Inline CSS

## 8. System Description

The system is a static website that allows users to view information
about the perfume brand, browse products, and access contact details.

## 9. Target Users

-   Customers interested in perfumes\
-   Friends and potential clients\
-   The business owner

## 10. Problems Encountered and Solutions

### Problem 1: Layout issues

Solution: Used CSS Flexbox and proper spacing.

### Problem 2: Images not loading

Solution: Fixed file paths and src attributes.

### Problem 3: Not responsive

Solution: Used media queries.

### Problem 4: Links not working

Solution: Corrected href values.

## 11. Future Improvements

-   Online ordering\
-   Payment system\
-   Admin dashboard\
-   Reviews\
-   Gallery

## 12. Conclusion

The website successfully provides a professional platform for promoting
Eunice Sununu's perfume business and applies core web development
concepts.
